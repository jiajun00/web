Tip #17
Multiple Menus
Visual QuickMenu allows for the simultaneous design of up to 10 menus. To add a new menu choose 'Modify' --> 'Add Menu'. You can mix and match horizontal and vertical menus, tree styles, different add-ons, etc.

New menus added with the interface are positioned by dragging and dropping the main container anywhere on the page. After publishing, or saving the menus and re-opening for editing, thier positions will be fixed.
