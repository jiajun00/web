Tip #2
Save Pure CSS or Hybrid Menus
Visual QuickMenu lets you choose your preferred menu structure. Choose between the classic UL / LI list based menu or our new more efficient Div / Anchor concept.

The classic UL / LI structured menus run with JavaScript turned off in the browser (IE7+, Firefox 1+, Mozilla, Opera 8+, Netscape 7+, Safari 2+)! With JavaScript enabled these menus take full advantage of older browsers and all the scripted add-ons and extended customization features offered by the visual interface.

Choose your ideal menu type under 'Settings' --> 'HTML Structure'.
