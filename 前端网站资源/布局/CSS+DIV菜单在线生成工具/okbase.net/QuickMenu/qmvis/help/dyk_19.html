Tip #19
Pure CSS Tree Menus
Changing the menu structure to 'Pure CSS' when saving a tree menu allows for zero script access to the entire structure. The pure CSS variant of a tree menu functions as a vertically oriented drop down menu.

Browsers with JavaScript enabled will see the full fledged tree menu with all add-ons and scripted features included.

To change the menu type select 'Settings' --> 'HTML Structure' from the main menu.
