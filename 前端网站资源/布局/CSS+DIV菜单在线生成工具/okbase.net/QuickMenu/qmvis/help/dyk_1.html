Tip #1
100% Pure CSS Cross Code Menus
Visual QuickMenu produces cross browser / cross code menus, capable of running on 100% Pure CSS alone, or JavaScript if it's present in the browser. A vast majority of browsers have their JavaScript enabled, allowing your visitors to take full advantage of the scripted feature enhancements offered by the QuickMenu interface.

Script enhanced menus retain the full benefits of true Pure CSS code with JavaScript disabled browsers, search engine spiders, screen readers, and validators.  Both cross code menu types pool from the same CSS and HTML structures without overhead.
