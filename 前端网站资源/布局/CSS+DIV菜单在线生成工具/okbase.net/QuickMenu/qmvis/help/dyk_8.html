Tip #8
Keep Subs on Top of Drop Down Lists in IE
Bugs in IE do not allow dynamic content to appear on top of select tags (drop down lists). To overcome this issue activate the 'Show Over Select Tags' add-on.

To activate this add-on choose 'Add-Ons --> Select Tag Fix (IE)' from the main menu.
