Tip #10
CSS Add-Ons [Imageless]
A host of CSS based imageless add-ons create rounded corners, drop shadows, tabs, bullets, and more. Theses add-ons use CSS and HTML without the need for any images.

Rounded Item and Bullet add-ons include hover and active settings, you can create a wide range of effects without the overhead of multiple images and save design time.

These add-ons are accessible in the tree or from the main add-ons menu.
