Tip #11
Quick Color Edits
The quick color editor is a fast way to change multiple colors in an existing menu from a single location. The quick color editor presents a list of each unique color in your menu, the number in parenthesis identifies how many styles will be changed.

The quick color editor may be accessed from the main menu (Settings --> Quick Color Edits).
