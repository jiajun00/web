Item Images
Any main or sub menu item may use custom images instead of text. Associated hover and active state images may also be specified.

To specify an image for any item first click on the menu item to make it active. In the visual interface expand individuals -- Item Extras -- Image. Here you can specify static, hover, and active images for the item.

If your design requires the image to fill the entire menu item you will need to remove any existing padding, borders, and margins currently applied to your item. After specifying an image you will be prompted to automatically zero these styles, or you can manually remove the values in the CSS styles section.

Active images only appear if the item has a child sub menu, hover images appear while the mouse is over the item.
