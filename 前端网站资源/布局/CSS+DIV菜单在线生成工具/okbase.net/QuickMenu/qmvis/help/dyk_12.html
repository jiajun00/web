Tip #12
Filtered CSS Styles
The first option in the settings tree is 'CSS Styles [filtered]'. This interface affords quick access to all the basic CSS settings for the menu.

To see settings that do not currently contain a value, check the 'add' option under the 'values' title.

The filtered settings are based on the styles in the 'CSS Styles [unfiltered]' section. Add-on styles and individual CSS settings are not filtered here.
