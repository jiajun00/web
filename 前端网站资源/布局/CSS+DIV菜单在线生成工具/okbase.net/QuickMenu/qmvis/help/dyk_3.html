Tip #3
Active Item CSS Styles
QuickMenu allows for two types of interactive item styles, the commonly known hover styles are displayed when the mouse is over the item. Active styles are displayed while the items sub menu is visible.

Many designs will require certain colors and styles to be applied to both the hover and active options for your items. If a menu item does not own a sub menu, its active styles will not be able to display.
