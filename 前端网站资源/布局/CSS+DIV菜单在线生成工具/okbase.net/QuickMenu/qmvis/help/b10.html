Visual QuickMenu
Visual Interface Compatibility (IE, Firefox, Safari, and Netscape)
The Visual QuickMenu Interface is compatible with the main stream modern browsers on all platforms.  The menus produced will run in many older browser versions and low use browsers such as Opera and Konqueror.

Safari 3.x supports the visual interface and runs the software very fast, Mac users can also use Firefox to run the visual tool.  On Windows, Safari 3, Firefox 2+, and IE 6 and 7 are supported.  We recommend using Safari or Firefox on windows, IE is many multiples slower than these browser when it comes to executing style changes and you will likely notice delays during visual design in the IE browser. Netscape 9.x on any platform is also supported.

The visual interface is virtually identical in all the supported browsers with only a few minor exceptions.  Safari will copy publish content in RTF format on windows, meaning you will need to paste this into a non-plain text environment such as an HTML editor or word.  Safari may also convert local relative file references to fully qualified paths when opening a menu, for this reason we recommend using fully qualified http://... paths in Safari.
